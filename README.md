# REDA INVESTISMENT

تطبيق ويب متكامل لإدارة المستثمرين.

## المتطلبات الأساسية
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

## خطوات التثبيت والتشغيل محلياً

1. **تثبيت الحزم (Dependencies):**
   قم بفتح موجه الأوامر (Terminal) في مجلد المشروع وقم بتشغيل:
   ```bash
   npm install
   ```

2. **تهيئة قاعدة البيانات وإضافة البيانات الافتراضية (Seeding):**
   لتجهيز قاعدة البيانات بحساب الإدارة والمستثمرين الوهميين، قم بتشغيل الأمرين:
   ```bash
   npx prisma db push
   npm run db:seed
   ```
   *ملاحظة: إذا لم يعمل `npm run db:seed`، يمكنك تشغيله مباشرة عبر: `node prisma/seed.js`*

3. **تشغيل خادم التطوير (Development Server):**
   ```bash
   npm run dev
   ```

4. **الوصول إلى التطبيق:**
   افتح المتصفح على الرابط: [http://localhost:3000](http://localhost:3000)

## التقنيات المستخدمة
- **Next.js (App Router)** للواجهات (Frontend) والواجهات البرمجية (Backend APIs).
- **Prisma + SQLite** لإدارة قاعدة البيانات محلياً بملف واحد `.db`.
- **Vanilla CSS (CSS Modules)** للتصميم مع دعم كامل للغة العربية (RTL).
- **JSON Web Tokens (jose)** للمصادقة وتأمين الجلسات محلياً.
- **bcryptjs** لتشفير كلمات المرور.
